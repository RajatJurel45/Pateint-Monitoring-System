# AI-based Patient Monitoring Web Application

This project is a next-generation, AI-powered patient-monitoring system that:

- Streams real-time vitals from medical devices or simulators
- Displays live waveforms
- Triggers intelligent anomaly alerts
- Predicts future vitals using deep learning (LSTM/GRU/Transformers)
- Provides a clean, modern, hospital-ready dashboard for doctors and nurses

Tech stack:
- Frontend: Next.js, React, Tailwind CSS
- Backend (ingestion & API): FastAPI, PostgreSQL/TimescaleDB, Redis, Kafka/MQTT
- ML/AI: PyTorch
- Infra: Docker, Kubernetes, Helm, GitHub Actions (CI/CD)
